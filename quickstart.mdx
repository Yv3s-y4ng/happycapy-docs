# 5 分钟上手

## 第一步：注册账号

访问 [happycapy.ai](https://happycapy.ai)，注册你的账号。

## 第二步：进入工作台

登录后你会看到一个对话界面，这就是你的 AI 工作台。

你的工作台已经预装了：
- AI 助手（基于 Claude）
- 96+ 种专业技能
- 图片/视频/文档等各种工具

## 第三步：给 AI 下指令

直接用中文描述你想做什么，比如：

```
帮我做一个产品介绍的落地页，主题色用蓝色，
包含产品特点、价格方案和联系方式
```

AI 会：
1. 理解你的需求
2. 自动选择合适的工具和技能
3. 一步步执行
4. 把成品交给你

## 第四步：查看和下载结果

任务完成后，成品会出现在对话中。你可以：
- 直接预览（网页、图片等）
- 下载文件（PDF、图片、视频等）
- 要求 AI 修改（"把标题字号再大一点"）

<Tip>
如果结果不满意，直接告诉 AI 哪里要改，它会帮你调整。
</Tip>

## 常见用法

### 做图片
```
帮我用 FLUX 模型生成一张赛博朋克风格的城市夜景
```

### 做视频
```
根据这段文字生成一个 15 秒的短视频
```

### 写文档
```
帮我写一份项目周报，包含本周完成的事项和下周计划
```

### 分析数据
```
分析这个 CSV 文件，做出销售趋势图表
```

### 做 PPT
```
把这篇文章的核心观点做成 10 页 PPT
```
