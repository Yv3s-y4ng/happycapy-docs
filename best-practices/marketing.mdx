Creative directions:

Output in CSV format with columns:

## Advice for Marketers

### 1. Identify Repetitive Tasks with API Support

Many marketing platforms have APIs that enable automation:

  <Card title="Ad Platforms" icon="bullhorn">
    Google Ads, Meta Ads, LinkedIn Ads
  </Card>

  <Card title="Design Tools" icon="pen-ruler">
    Figma, Canva

    Google Analytics, Mixpanel
  </Card>

<Tip>
  These platforms are excellent candidates for automation workflows.
</Tip>

### 2. Split into Specialized Subtasks

Don't ask for everything at once. Break complex campaigns into steps:

<AccordionGroup>
  <Accordion icon="heading" title="Task 1: Generate Headlines">
    ```
    Generate 50 headline variations for our summer campaign
    ```
  </Accordion>

  <Accordion icon="paragraph" title="Task 2: Generate Descriptions">
    ```
    Generate description variations matching these headlines
    ```
  </Accordion>

  <Accordion icon="table" title="Task 3: Combine and Format">
    ```
    Combine headlines and descriptions into all valid
    permutations, output as CSV
    ```
  </Accordion>
</AccordionGroup>

<Note>
  This approach makes debugging and optimization easier.
</Note>

### 3. Plan Thoroughly Before Implementing

<Columns cols={2}>
  <div>
    **Plan First:**
    - Map out the complete workflow
    - Identify data sources
    - Define output format
    - Consider edge cases
  </div>
  <div>
    **Then Execute:**
    - Implement step by step
    - Test each stage
    - Refine based on results
    - Don't ask for one-time solution
  </div>
</Columns>

### 4. Build a Memory System to Learn Across Iterations

Build a knowledge base from your campaigns:

```
Record the results from our last A/B test:

Test period: Jan 1-15, 2025
Variants tested:
- "Free Trial" messaging: 5.2% CTR, 3.1% conversion
- "Limited Time Offer" messaging: 4.1% CTR, 2.4% conversion

Finding: "Free Trial" messaging performs 25% better in conversion rate.

For our next campaign, generate variations emphasizing
"free trial" messaging with similar positioning.
```

<Tip>
  Let AI learn from your data to improve future campaign performance.
</Tip>

## Real-World Examples

### Example 1: Email Campaign Variations

```
Create 10 email subject line variations for our product launch:

Product: Happycapy Max Plan
Launch date: March 15
Target: Existing Free users

Themes to explore:
- Urgency (launch day only discount)
- Curiosity (new features reveal)
- Social proof (what other users love)
- Value proposition (ROI calculations)

Requirements:
- 60 characters or less
- No spammy words
- A/B test friendly (clear variants)
- Include emoji suggestions
```

### Example 2: Landing Page Copy

```
Write landing page copy for our new AI video generation feature:

Target audience: Content creators, marketers

Key benefits:
- Generate videos from text prompts
- 40+ AI models available
- No video editing skills required
- Export in multiple formats

Sections needed:
1. Hero headline + subheadline
2. 3 feature highlights (headline + 2 sentence description each)
3. Social proof section (how to frame testimonials)
4. FAQ section (5 common questions)
5. CTA variations (3 different approaches)

Tone: Professional but approachable, focus on creative empowerment
```

### Example 3: Social Media Content Calendar

```
Create a 2-week social media content calendar:

Platforms: Twitter, LinkedIn, Instagram
Product: Happycapy
Goals: Drive signups, showcase use cases, build community

Content mix:
- 40% Educational (tips, tutorials, best practices)
- 30% Use cases (customer stories, examples)
- 20% Product updates (features, improvements)
- 10% Community (user content, engagement)

For each post provide:
- Platform
- Post copy
- Image suggestion

- Relevant hashtags
- Link to include

Output as spreadsheet format.
```

## Best Practices for Marketing Automation

### Version Control Your Copy

<Tip>
  Keep track of winning variations and build a swipe file of high-performing copy. Reference these in future prompts:

  ```
  Generate new variations similar to our best-performing ad:
  [Paste winning copy]

  Maintain the same tone and structure but explore new angles.
  ```
</Tip>

### Test Systematically

Create structured A/B tests:

```
Set up A/B test variants:

Control: Current headline "Boost Your Productivity"
Test variables:
- Headline variations (5 options)
- CTA button text (3 options)
- Image style (2 options)

Generate test matrix showing all combinations and
predictions about which might perform best based on
marketing psychology principles.
```

### Automate Reporting

```
Create an automated weekly marketing report:

Data sources:
- Google Ads performance (via API or CSV export)
- Facebook Ads data
- Website analytics
- Email campaign metrics

Report sections:
1. Executive Summary (key metrics vs. targets)
2. Channel Performance (week over week)
3. Top Performing Content
4. Budget Pacing
5. Recommendations for Next Week

Format as PDF with charts and tables.
Schedule this to run every Monday at 9 AM. (Pro/Max plans)
```
