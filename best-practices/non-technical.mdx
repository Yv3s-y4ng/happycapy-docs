## Who This is For

Finance, Legal, Marketing, Operations, Product Managers, and other professionals without programming backgrounds

## Core Scenarios

### Scenario 1: Automated Data Workflow

**Real Case:** The finance team uses plain text to describe workflows and lets AI automatically perform complex data processing tasks.

#### How to Do It

<AccordionGroup>
  <Accordion icon="comment" title="Step 1: Describe Your Workflow in Natural Language">
    Just explain what you want to accomplish:

    ```
    At the beginning of every month, please help me:
    1. Read last month's sales data from Excel table
    2. Calculate the growth rate for each region
    3. Generate comparison chart
    4. Create a report with key metrics
    ```
  </Accordion>

  <Accordion icon="question" title="Step 2: Happycapy Will Ask for Required Input">
    Happycapy will actively ask questions like:
    - "Please provide Excel file"
    - "Which month's data need to be compared?"
  </Accordion>

  <Accordion icon="calendar" title="Step 3: Save as Automated Process (Pro/Max Plan)">
    - Set up scheduled tasks to be executed automatically every month
    - No need to re-describe each time
  </Accordion>
</AccordionGroup>

### Scenario 2: Generate Marketing Materials Quickly

**Real Case:** Growth marketing team used Happycapy to generate hundreds of ad copy variations in 15 minutes.

#### How to Do It

Provide detailed requirements for batch creation:

```
Help me create 20 Google Ads ad variations:
- Product: AI collaboration tool Happycapy
- Target audience: small and medium business owners
- Selling points: zero technical threshold, improved work
  efficiency, cloud operation
- Title limit: within 30 characters
- Description limit: within 90 characters
```

#### What Happycapy Will Do

1. Generate multiple creative directions
2. Automatically check character limits
3. Output as CSV that can be directly imported into advertising platforms

**Time savings: from 2 hours → 15 minutes**

## Advice for Non-Technical People

### 1. Start with Simple Tasks to Build Confidence

<AccordionGroup>
  <Accordion icon="image" title="First Try: Basic Creation">
    Try a simple task: "Generate a poster for me"
  </Accordion>

  <Accordion icon="chart-line" title="Then Try: Data Analysis">
    Move to data tasks: "Analyze this data and make a graph"
  </Accordion>

  <Accordion icon="diagram-project" title="Finally Try: Complex Workflows">
    Tackle more complex multi-step workflows
  </Accordion>
</AccordionGroup>

### 2. Describe Your Needs Like Chatting with an Assistant

<Columns cols={2}>
  <div>
    **Don't Think:**

    "What tool should I use?"
    "How do I program this?"
  </div>
  <div>
    **Just Say:**

    "What result do I want?"
    "What's my end goal?"
  </div>
</Columns>

<Tip>
  Happycapy will automatically select the appropriate tools and models for you.
</Tip>

### 3. Save Successful Workflows

When you find a prompt that works well:

1. **Record effective prompts** in your notes
2. **For repeated tasks**, directly copy the last description
3. **Pro/Max users** can set up automation to run tasks on a schedule

### 4. Don't be Afraid of "Dumb" Questions

<Note>
  - Happycapy won't judge you
  - Trial and error is part of learning
  - Each conversation is independent and can be restarted at any time
</Note>

## Real-World Examples

### Finance: Monthly Budget Report

```
Help me create a monthly budget vs. actual spending report:

1. Read budget.xlsx and spending_january.xlsx
2. Calculate variance for each category
3. Highlight categories that are over budget in red
4. Create a visual chart showing:
   - Budget vs. Actual comparison
   - Percentage of budget used
5. Generate a one-page executive summary
```

### Legal: Contract Analysis

```
Analyze these 5 vendor contracts and create a comparison table:

[Upload contract files]

Please extract and compare:
- Contract duration
- Payment terms
- Liability clauses
- Termination conditions
- Key obligations

Create a summary table that I can use to make a decision.
```

### Operations: Task Management

```
Process my weekly team meeting notes:

[Paste meeting transcript]

Please:
1. Identify all action items
2. Assign to the person mentioned in context
3. Extract due dates (or suggest reasonable dates)
4. Create a table with columns: Task, Owner, Due Date, Status
5. Export as CSV for importing into our project management tool
```

### HR: Candidate Screening

```
Help me screen these 20 resumes:

[Upload resume files]

Job requirements:
- 3+ years experience in project management
- PMP certification preferred
- Experience with Agile methodologies
- Strong communication skills

Please:
1. Rank candidates by qualification match
2. Highlight key strengths and gaps for each
3. Create a shortlist of top 5 candidates
4. Generate interview questions based on their backgrounds
```

## Tips for Success

### Start Small and Scale Up

Don't try to automate everything at once. Instead:

1. **Pick one repetitive task** that takes 30+ minutes per week
2. **Automate that single task** with Happycapy
3. **Refine the workflow** over 2-3 iterations
4. **Move to the next task** once the first is working smoothly

### Build Your Prompt Library

<Tip>
  Create a document with your best-performing prompts organized by task type. This becomes your personal automation toolkit.
</Tip>

### Use Templates

Once you've created a successful workflow:

- Save the prompt structure
- Replace specific details (dates, filenames, etc.) each time
- Share successful templates with your team

### Ask for Explanations

If you want to understand what Happycapy is doing:

```
Please explain each step you're taking to process this data
and why you chose this approach.
```

This helps you learn and improve future prompts.
