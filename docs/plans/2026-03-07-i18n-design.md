# Combined Design: i18n + User Cases — HappyCapy Documentation

**Date**: 2026-03-07
**Status**: Approved

## Part 1: i18n

### Goal

Add Simplified Chinese (zh) and Traditional Chinese (zh-Hant) to the existing English documentation site at happycapy.mintlify.app.

### Decisions

- **Approach**: English at root, zh/ and zh-Hant/ folders for translations
- **Default language**: English (en)
- **Languages**: en, zh, zh-Hant
- **Nav labels**: Fully translated per language (tabs, groups, page titles)
- **Images**: Shared `/images/` directory across all languages
- **Translation source**: Notion-exported HTML files at `happycapy 多语言/extracted/`

### docs.json Navigation

Uses `navigation.languages` array with `tabs` nested inside each language entry.

- EN tabs: Getting Started, Feature Guides, Best Practices, Integrations, Use Cases
- zh tabs: 快速入门, 功能指南, 最佳实践, 集成, 用户案例
- zh-Hant tabs: 快速入門, 功能指南, 最佳實踐, 整合, 使用案例

## Part 2: User Cases Section

### Goal

Add a "Use Cases" tab as a community showcase for user-submitted stories. Team receives user submissions, formats them into MDX pages, and publishes.

### Decisions

- **Placement**: New 5th top-level tab
- **Organization**: By scenario categories (Coding, Design, Content Creation, Data & Analysis, Other)
- **Initial content**: Overview page only (framework). Cases added as users submit.
- **Growth model**: Each new submission = 1 new MDX file + 1 line added to docs.json nav per language

### Navigation Structure

```
Use Cases (tab)
  └── Use Cases (group)
        └── use-cases/index.mdx    -- Overview + how to submit
```

Categories will be added as groups when submissions come in:
- Coding
- Design
- Content Creation
- Data & Analysis
- Other

### Submission Template

Each user story page follows:

```mdx
---
title: "Username - Short Title"
description: "Brief scenario description"
---

## Background
Who the user is and what they needed.

## What They Did
The prompt or workflow they used in HappyCapy.

## Result
What HappyCapy produced, with screenshots if available.

## Key Takeaway
What others can learn from this case.
```

### Initial Files

- `use-cases/index.mdx` (EN overview)
- `zh/use-cases/index.mdx` (zh overview)
- `zh-Hant/use-cases/index.mdx` (zh-Hant overview)
