## Happycapy是什麼?

一臺**執行在瀏覽器裏的 Agent 原生計算機**。

由 Claude Code 驅動,為所有人設計。

不需要設定環境,不需要學命令列,也不需要技術背景。打開瀏覽器,說出你的需求,看著它協助你完成工作。

生成圖片、剪輯影片、處理資料、做網頁、寫論文...以前需要學十幾個軟體才能做的事,現在只需要說出來。

過去一年,我們看到 AI Agent 能做的事越來越多。但大部分 Agent 工具都是為開發者設計的——要會用命令列,要懂得設定環境,要知道怎麼處理安全問題。我們覺得,Agent 不應該只屬於懂技術的人。所以我們做了 happycapy。

## happycapy背後的故事

### 我們自己先變成了 AI native 團隊

2025 年 1 月,Claude Code 出來了。它改變了編碼的模式——開發者可以用自然語言寫程式碼,效率提升了好幾倍。

我們團隊用得很爽。但同時我們也在想:**為什麼 AI 的門檻還是這麼高?**

想用好 AI,要麼懂命令列、會配環境、學提示詞,要麼就只能停留在聊天窗口裏,很難發揮出 AI 的全部能力。為什麼不能像用電腦一樣簡單?

**AI 應該是平權的。**

不管你是什麼職業、什麼背景——每個人都應該平等地享受 AI 帶來的生產力提升。

所以我們給自己做了一個工具。用上之後,整個團隊的工作方式改變了。不懂技術的人也能自己寫程式碼、做網頁、處理資料。每個人都能把想法變成現實。

**每個人都成了超級個體。**

然後我們把這套工作方式產品化了,就是 happycapy。

### 為什麼是Capy(卡皮巴拉-Capybara-水豚)

水豚溫和、友善、與所有動物和諧相處。你能在網上看到水豚和猴子、鳥、鱷魚、甚至貓狗都能玩在一起。

這正是我們想要的:一個和所有人都能"相處"的 AI 工具。

**Happy + Capy = 快樂的水豚 = AI應該使人變得快樂而不是焦慮。**

我們希望大家用 happycapy 的時候,就像養一隻卡皮巴拉一樣——chill,沒有焦慮,迴歸到生活本身。

## 我們的願景

**讓所有人都用上 AI,自動化自己的工作流,減少人的重複性工作。**

**happycapy, HappyYou.**

我們不覺得 AI 是用來取代人的。我們覺得 AI 是用來讓人更像人的——把重複的、煩人的、消耗時間的事情交給 AI,把創意、判斷、決策留給自己。

你只需要專注於**做什麼**,而不是**怎麼做**。

不用糾結今天該用哪個模型,不用被每天 AI 新突破的新聞搞得焦慮。
