## 是什麼

Automation 按時間表自動向指定 Desktop 發送 prompt 並執行。你寫好 prompt，設定時間，Happycapy 自動跑——不需要手動操作。

## 建立 Automation

點擊側邊欄的 **Automations**，然後點 **Create Automation**。

<Frame>
  <img src="/images/features/automation-create.png" alt="建立 Automation 對話框" />
</Frame>

填寫：

- **Name** — 起個描述性名字（例如 "Morning summary"）
- **Desktop** — 在哪個 Desktop 裡執行
- **Session binding** — 綁定哪個 session（或主 session）
- **Schedule** — 每天定時，或按間隔（每 N 小時）。可以選星期幾。
- **Prompt** — Happycapy 要執行的內容
- **Run immediately** — 開啟後會立即執行一次

點 **Create** 完成。

## 範例

每天早上的新聞摘要：

```
抓取 AI、創業和產品設計領域的熱門新聞。
過濾掉標題黨，每條新聞用一句話總結。
生成 HTML 簡報寄到我的信箱。
```

每小時檢查 GitHub PR：

```
檢查 my-org/my-repo 所有 open 的 PR。
列出需要我 review 或有新留言的。
把結果發到當前 Desktop。
```

每週專案報告：

```
總結這個 Desktop 過去一週的所有工作。
列出已完成任務、未解決問題和下一步計畫。
生成報告儲存到 outputs/。
```

<Tip>
  Prompt 的執行方式和你在對話裡手動輸入完全一樣。你能在 session 裡做的事，Automation 都能定時做。
</Tip>
