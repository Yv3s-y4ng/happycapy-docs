## 第一步:打開 Happycapy

訪問 [happycapy.ai](https://happycapy.ai),註冊並登入後,你會看到主介面:

<Columns cols={3}>
  <div>
    **左側邊欄**
    - Desktops
    - Skills
    - Automations
    - Skill Store
  </div>
  <div>
    **對話區**
    - 與 Happycapy 對話
    - 查看生成的內容
    - 監控進度
  </div>
  <div>
    **檔案管理**
    - `uploads/` 資料夾
    - `outputs/` 資料夾
    - 上傳按鈕 (+)
  </div>
</Columns>

## 第一次對話

別緊張,就像和朋友聊天一樣。

在中間的輸入框裏,試試這句話:

```
畫一隻可愛的卡皮巴拉,漫畫風格
```

按回車,看著 Happycapy 為你創建圖片。

或者試試:

```
生成不同美術風格的可愛卡皮巴拉
```

Happycapy 會根據你的要求調整。你可以在 `outputs/` 資料夾中看到這些圖片並可以下載。版權完全屬於你。

## 處理檔案

直接拖曳檔案到對話視窗，或點擊對話輸入框旁的 **+** 按鈕從本機選擇檔案。上傳的檔案會放進沙箱，Happycapy 可以立即讀取和處理。

<Note>
  支援的格式：圖片（JPG/PNG）、文件（PDF/Word/Excel）、影片（MP4）、資料檔案（CSV/JSON）、程式碼（所有語言）。單個檔案最大 100MB，支援批次上傳。
</Note>

Happycapy 產生的檔案——圖片、處理後的文件、匯出的資料——都會保存在沙箱裡。點擊左側邊欄的 **Files** 隨時瀏覽、搜尋和下載。

## 快速上手的例子

<AccordionGroup>
  <Accordion icon="image" title="案例 1:生成圖片並下載">
    最簡單的例子,10 秒上手。

    **第 1 步:說出你的需求**

    在輸入框輸入:
    ```
    生成一張圖片:可愛的卡皮巴拉在草地上曬太陽
    ```

    **第 2 步:等待生成**

    圖片會在約 10-15 秒內顯示在對話區,自動保存在 `outputs/` 資料夾。

    **第 3 步:下載到本地**

    - **方法 1**:右側檔案列表找到圖片,右鍵下載
    - **方法 2**:直接說「把這張圖片打包給我下載」

    <Tip>
      不滿意?直接說「再來一張,風格更卡通一點」,Happycapy 會繼續調整。
    </Tip>
  </Accordion>

  <Accordion icon="file-arrow-up" title="案例 2:上傳檔案並處理">
    學會上傳檔案,Happycapy 能協助你處理各種檔案。

    **第 1 步:上傳檔案**

    - **方法 1**:直接拖拽檔案到對話窗口
    - **方法 2**:點選輸入框左邊的 + 號,選擇檔案

    **第 2 步:告訴 Happycapy 要做什麼**

    ```
    請幫我整理這份 Excel:
    1. 刪除重複資料
    2. 按日期排序
    3. 生成一份乾淨的新表格
    ```

    **第 3 步:等待處理完成**

    Happycapy 會讀取檔案、處理資料、生成新檔案,並告訴你改了什麼。處理好的檔案會保存在 `outputs/` 資料夾。
  </Accordion>
</AccordionGroup>
