## 常見問題

<AccordionGroup>
  <Accordion icon="shield-check" title="我的資料安全嗎?隱私會被泄露嗎?">
    你的隱私和資料安全是我們的首要關注:

    ### 本地優先
    你的檔案只有在你主動上傳時纔會到達雲端沙盒環境,不會影響你的本地系統。

    ### 沙盒隔離
    所有程式碼在獨立的沙盒環境中執行,與你的主系統完全隔離,確保安全無虞。

    ### 透明可控
    AI 的每一步操作都清晰可見。你可以隨時查看、暫停或修改。

    ### 權限自主
    連接第三方服務(GitHub、Notion 等)時,所有權限由你授權,隨時可以撤銷。
  </Accordion>

  <Accordion icon="file-arrow-up" title="可以處理多大的檔案?">
    HappyCapy 可以處理各種大小的檔案,但處理大檔案時,我們推薦這樣做:

    ### 小檔案 (< 50MB)
    直接上傳到對話框即可。

    ### 大檔案 (> 50MB)
    為了更好的性能:
    1. 先上傳到 GitHub
    2. 讓 HappyCapy 從雲端下載

    這樣更快更穩定。

    <Note>
      **GitHub 集成提示**:生成一個 GitHub Token,HappyCapy 就能協助你下載、上傳、管理檔案。詳細說明請參見我們的 GitHub 集成指南。
    </Note>
  </Accordion>

  <Accordion icon="code" title="使用 HappyCapy 需要編程知識嗎?">
    **不需要編程知識!**

    HappyCapy 為所有人設計,無論技術背景如何。你用自然語言與 HappyCapy 溝通,就像和朋友聊天一樣。

    不過,如果你有編程知識,你可以:
    - 給出更技術性的指示

  </Accordion>

  <Accordion icon="clock" title="檔案會在工作區保留多久?">
    你的 HappyCapy 工作區中的檔案是持久的:

    - **上傳的檔案**:保留在 `uploads/` 直到你刪除
    - **生成的輸出**:保留在 `outputs/` 直到你刪除
    - **Desktop 會話**:自動保存,隨時可訪問

    你對檔案有完全控制權,可以隨時刪除。
  </Accordion>
