## 適用場景

有些開發場景下，你可能需要使用本地 IDE：

- 大型專案開發（數百個檔案）
- 使用已配置好的本地開發環境
- 需要特定的付費擴充或企業工具
- 習慣本地 IDE 的快捷鍵與工作流程

好消息：透過 **VS Code Remote Tunnels**，可以輕鬆連接到 happycapy 雲端沙箱！

## 3 步完成連接

<Steps>
  <Step title="在 happycapy 啟動 Tunnel">
    在 happycapy 對話框中輸入：

    ```
    請幫我啟動 VSCode Remote Tunnel（全局模式）--name happycapy，並展示 GitHub 認證連結與驗證碼
    ```

    happycapy 會自動安裝並啟動 Tunnel，然後回傳：

    <Note>
    **記下這兩個資訊：**
    - 授權頁面：`https://github.com/login/device`
    - 授權碼：`XXXX-XXXX`（8 位代碼）
    </Note>
  </Step>

  <Step title="GitHub 授權">
    在瀏覽器中完成授權（約 1 分鐘）：

    1. 開啟 [https://github.com/login/device](https://github.com/login/device)
    2. 輸入 happycapy 提供的授權碼
    3. 點擊 **Authorize**

    <Tip>
    成功標誌：瀏覽器顯示 "Success! You may close this page."
    </Tip>
  </Step>

  <Step title="從本地 VS Code 連接">
    <Accordion title="安裝 Remote - Tunnels 擴充">
      1. 打開本地 VS Code
      2. 按 `Ctrl + Shift + X` 開啟擴充面板
      3. 搜索 **Remote - Tunnels**（Microsoft 官方）
      4. 點擊 **Install**
    </Accordion>

    <Accordion title="連接 Tunnel">
      1. 按 `F1` 開啟命令面板
      2. 輸入：`Remote-Tunnels: Connect to Tunnel`
      3. 登入 GitHub（首次需要）
      4. 選擇 `happycapy`
      5. 等待連接建立

      <Tip>
      成功標誌：左下角顯示綠色 `>< Remote [happycapy]`
      </Tip>
    </Accordion>
  </Step>
</Steps>

## 開始使用

連接成功後，你可以像在本地一樣使用 happycapy 沙箱：

### 打開遠端資料夾

<Steps>
  <Step title="點擊 'Open Folder...'">
    從 VS Code 檔案選單或歡迎畫面
  </Step>

  <Step title="輸入資料夾路徑">
    常用路徑：
    - `/home/node` - 你的主目錄
    - `/workspace` - 你的工作區目錄
  </Step>

  <Step title="信任工作區">
    出現提示時點擊「是的，我信任作者」
  </Step>
</Steps>

### 打開終端機

按 `Ctrl + \`` (反引號) 開啟遠端終端機。

<Tip>
**驗證連接：** 執行 `pwd` 和 `whoami` 確認你在雲端沙箱中
</Tip>

## 快速參考

### happycapy Prompt 速查

| 操作 | Prompt |
|------|--------|
| 啟動 Tunnel | `請幫我啟動 VSCode Remote Tunnel（全局模式）--name happycapy，並展示 GitHub 認證連結與驗證碼` |
| 檢查狀態 | `VSCode Tunnel 還在運行嗎？` |
| 停止服務 | `請停止 VSCode Tunnel 服務` |
| 重新啟動 | `請重新啟動 VSCode Tunnel` |

### VS Code 快捷鍵

| 操作 | 快捷鍵 |
|------|--------|
| 打開資料夾 | `Ctrl + K`, 然後 `Ctrl + O` |
| 打開終端機 | `Ctrl + \`` |
| 命令面板 | `F1` 或 `Ctrl + Shift + P` |
| 快速開啟 | `Ctrl + P` |
| 中斷連接 | 點擊左下角 `>< Remote` 標誌 |

## 疑難排解

<AccordionGroup>
  <Accordion title="找不到我的 tunnel？">
    1. 在 happycapy 中問：「VSCode Tunnel 還在運行嗎？」
    2. 如果沒有運行，重新執行啟動 prompt
    3. 檢查本地網路連線
    4. 重新登入 GitHub：
       - 按 `F1` → `Remote-Tunnels: Sign out`
       - 然後重新連接並登入
  </Accordion>

  <Accordion title="連接逾時">
    嘗試以下步驟：
    1. 檢查網路連線
    2. 暫時關閉 VPN（如果正在使用）
    3. 在 happycapy 中重啟 Tunnel
    4. 更新 VS Code 到最新版本
  </Accordion>

  <Accordion title="無法儲存檔案">
    1. 檢查磁碟空間：在遠端終端機執行 `df -h`
    2. 檢查檔案權限：執行 `ls -la`
    3. 嘗試重新連接 Tunnel
  </Accordion>

  <Accordion title="擴充無法運作">
    某些擴充需要安裝在遠端伺服器上：
    1. 開啟擴充面板 (`Ctrl + Shift + X`)
    2. 尋找有「Install in SSH: happycapy」按鈕的擴充
    3. 點擊安裝到遠端伺服器
  </Accordion>
</AccordionGroup>
