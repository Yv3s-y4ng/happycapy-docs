## 常见问题

<AccordionGroup>
  <Accordion icon="shield-check" title="我的数据安全吗?隐私会被泄露吗?">
    你的隐私和数据安全是我们的首要关注。以下是我们如何保护你:

    ### 本地优先
    你的文件只有在你主动上传时才会到达云端沙盒环境,不会影响你的本地系统。

    ### 沙盒隔离
    所有代码在独立的沙盒环境中运行,与你的主系统完全隔离,确保安全无虞。

    ### 透明可控
    AI 的每一步操作都清晰可见。你可以随时查看、暂停或修改操作。

    ### 权限自主
    连接第三方服务(GitHub、Notion 等)时,所有权限由你授权,随时可以撤销。
  </Accordion>

  <Accordion icon="brain" title="Happycapy 支持哪些 AI 模型?">
    Happycapy 通过 gateway 接入多家 AI 服务商：

    - **Claude** — Opus 4.6、Sonnet 4.6、Haiku 4.5
    - **Google** — Gemini 3 Pro Preview、Gemini 3 Pro Image
    - **xAI** — Grok 3
    - **MiniMax** — M2.7

    <Tip>
      模型列表会随着新服务商的接入持续更新。你不需要管理 API key，gateway 自动处理路由。
    </Tip>
  </Accordion>

  <Accordion icon="file-arrow-up" title="可以处理多大的文件?">
    Happycapy 可以处理各种大小的文件,但我们推荐以下最佳实践:

    ### 小文件 (< 50MB)
    直接上传到对话框即可即时处理。

    ### 大文件 (> 50MB)
    为了更好的性能:
    1. 先上传到 GitHub
    2. 让 Happycapy 从云端下载

    这种方式对于大文件更快更稳定。

  </Accordion>

  <Accordion icon="dollar-sign" title="价格方案是什么?">
    Happycapy 提供灵活的方案以适应不同需求:

    ### 免费套餐
    有限的月度使用额度,让你探索 Happycapy 的功能。

    ### Pro 套餐
    为需要更多容量的普通用户提供慷慨的 token 限制。

    ### Max 套餐

    访问 [happycapy.ai/pricing](https://happycapy.ai/pricing) 查看当前价格详情。
  </Accordion>

  <Accordion icon="globe" title="我可以在移动设备上使用 Happycapy 吗?">
    可以!Happycapy 完全在浏览器中运行,所以你可以从以下设备访问:

    - 台式电脑 (Windows, Mac, Linux)
    - 平板电脑 (iPad, Android 平板)
    - 智能手机 (iOS, Android)

    界面会适应你的屏幕大小,尽管一些复杂任务在大屏幕上更容易操作。
  </Accordion>

  <Accordion icon="language" title="Happycapy 支持哪些语言?">
    Happycapy 支持多种对话语言:

    - 英语
    - 中文(简体和繁体)
    - 日语
    - 韩语
    - 西班牙语
    - 法语
    - 德语
    - 等等

    只需用你喜欢的语言与 Happycapy 交谈,它会相应地回应。
  </Accordion>

  <Accordion icon="plug" title="我可以将 Happycapy 与其他工具集成吗?">
    Happycapy 基于 Claude Code 运行，支持大部分 MCP server、API 和 CLI 工具。常见集成包括：

    - **GitHub**：仓库管理、文件上传、版本控制
    - **Notion**：文档管理和协作
    - **Gmail**：邮件处理和自动化
    - **Google Drive**：云存储集成

    所有集成都需要你的授权。你可以在对话中直接让 Capy 帮你配置。
  </Accordion>

  <Accordion icon="code" title="我需要懂编程才能使用 Happycapy 吗?">
    **不需要编程知识!**

    Happycapy 是为所有人设计的,无论技术背景如何。你用自然语言与 Happycapy 交流,就像和朋友聊天一样。

    不过,如果你确实有编程知识,你可以:
    - 给出更技术性的指令
    - 要求特定的实现方式
    - 查看和修改生成的代码

    Happycapy 会适应你的专业水平。
  </Accordion>

  <Accordion icon="clock" title="文件在我的工作区会保留多久?">
    你的 Happycapy 工作区中的文件是持久保存的:

    - **上传的文件**: 保留在 `uploads/` 中,直到你删除它们
    - **生成的输出**: 保留在 `outputs/` 中,直到你删除它们
    - **Desktop 会话**: 自动保存,随时可访问

    你对文件有完全的控制权,可以随时删除它们。
  </Accordion>
