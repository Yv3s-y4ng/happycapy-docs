## 什么是 Skills?

Skills 是 HappyCapy 的"能力插件",就像给 AI 安装不同的专业技能。它们不仅仅是简单的文本指令——Skills 可以:

- **调用外部 API**: 连接 GitHub、Notion、Google 等第三方服务,实现跨平台数据同步
- **运行 Python/JavaScript 脚本**: 执行复杂的数据处理、图像生成、视频编辑等任务
- **使用 MCP 协议** (Model Context Protocol): 像乐高积木一样组合各种工具能力

<Note>
Skills 是下一代软件形态:轻量(几 KB 文本文件)、开源(GitHub 共享)、易编辑(5 分钟定制)。不需要安装臃肿的应用,一个 Skill 就能获得专业功能。
</Note>

## 如何使用 Skills

使用 Skills 超简单,HappyCapy 会根据你的需求自动调用。

### 方式 1:自然语言(推荐)

直接说你想做什么,HappyCapy 自动选择合适的 Skill。

```
生成一张卡皮巴拉的插画
```

HappyCapy 会自动调用图片生成 Skill。

<Tip>
大部分时候,用自然语言就够了。HappyCapy 会自动识别并调用最合适的 Skill。
</Tip>

### 方式 2:手动选择

可以通过点击对话框的 **Skills** 按钮,或者输入斜杠命令 `/` 来唤起 Skill 选择菜单,浏览已安装的 Skills 并手动选择使用。

## 如何查找 Skills

Skills 是开源的,全球开发者在贡献各种实用技能。你可以在这些地方找到:

### 方式 1:直接问 HappyCapy(最简单)

```
帮我找一个数据处理的Skill
```

HappyCapy 会使用内置的 `find-skills` 功能,自动搜索开源社区,向你推荐最合适的 Skills。

### 方式 2:在社区平台搜索

## 如何创建 Skills

创建 Skill 很简单,就是用自然语言描述你想要的功能。

### 完整案例:求职简历优化器

<AccordionGroup>
  <Accordion icon="lightbulb" title="痛点">
    "投了 100 份简历没回应,不知道怎么改。"
  </Accordion>

  <Accordion icon="hammer" title="解决方案">
    创建一个专门优化简历的 Skill。

    ```
    帮我创建一个 Skill:

    名称:求职简历优化器

    功能:
    输入:我的简历 + 目标岗位 JD

    处理:
    1. 分析 JD 的关键词和要求
    2. 对比简历,找出匹配点和不足
    3. 优化措辞(更符合岗位)
    4. 量化成果(用数据说话)
    5. 调整排版(突出重点)
    6. 生成 3 个版本:保守版/平衡版/激进版

    输出:
    - 3 个优化后的简历
    - 每个版本的使用建议
    - 面试准备要点

    并帮我测试一下!
    ```
  </Accordion>

  <Accordion icon="check" title="使用你的新 Skill">
    创建后,这个 Skill 就保存在你的 Skill Store 了。以后每次投简历,只需上传新的 JD,Skill 自动优化!
  </Accordion>
</AccordionGroup>

### 创建步骤

直接告诉 HappyCapy 你想创建什么技能!描述:
- **名称**: 给你的 Skill 起个名字
- **功能**: 需要什么输入
- **处理**: 如何工作
- **输出**: 应该产生什么

<Tip>
你还可以改编别人的技能!如果觉得某个 Skill 不够完美,可以基于它进行修改。站在巨人的肩膀上,让工具更贴合你的需求。
</Tip>
