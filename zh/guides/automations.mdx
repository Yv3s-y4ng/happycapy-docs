## 是什么

Automation 按时间表自动向指定 Desktop 发送 prompt 并执行。你写好 prompt，设定时间，Happycapy 自动跑——不需要手动操作。

## 创建 Automation

点击侧边栏的 **Automations**，然后点 **Create Automation**。

<Frame>
  <img src="/images/features/automation-create.png" alt="创建 Automation 对话框" />
</Frame>

填写：

- **Name** — 起个描述性名字（比如 "Morning summary"）
- **Desktop** — 在哪个 Desktop 里运行
- **Session binding** — 绑定哪个 session（或主 session）
- **Schedule** — 每天定时，或按间隔（每 N 小时）。可以选星期几。
- **Prompt** — Happycapy 要执行的内容
- **Run immediately** — 打开后会立即运行一次

点 **Create** 完成。

## 示例

每天早上的新闻摘要：

```
抓取 AI、创业和产品设计领域的热门新闻。
过滤掉标题党，每条新闻用一句话总结。
生成 HTML 简报发到我的邮箱。
```

每小时检查 GitHub PR：

```
检查 my-org/my-repo 所有 open 的 PR。
列出需要我 review 或有新评论的。
把结果发到当前 Desktop。
```

每周项目报告：

```
总结这个 Desktop 过去一周的所有工作。
列出已完成任务、未解决问题和下一步计划。
生成报告保存到 outputs/。
```

<Tip>
  Prompt 的执行方式和你在对话里手动输入完全一样。你能在 session 里做的事，Automation 都能定时做。
</Tip>
