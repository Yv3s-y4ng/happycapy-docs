## 是什么

Agent 是自定义的 AI 角色，拥有独立的身份、记忆和技能组合。默认 Agent 是 **Happycapy**。你可以为不同工作流创建不同的 Agent——研究助手、代码审查员、写作搭档——各有各的性格和上下文。

## Agent 文件

</Frame>

每个 Agent 通过一组 markdown 文件来定义自己：

- **SOUL.md** — 核心价值观和原则
- **USER.md** — 关于你（被服务者）的信息
- **IDENTITY.md** — Agent 的角色和性格
- **MEMORY.md** — 跨会话的持久记忆
- **AGENTS.md** — 主指令文件，把所有东西串起来

每次开始对话时，Agent 按顺序读取这些文件——先知道自己是谁，再开始工作。

## 配置 Agent

不用手写这些文件。在侧边栏的 **Agents** 里创建新 Agent，然后开一个对话，选定这个 Agent，直接说：

```
帮我配置这个 Agent。
```

Happycapy 会问你需要什么，然后自动生成所有配置文件。描述角色、你的偏好、你希望它记住的东西——Happycapy 搞定剩下的。

## 模型和技能

在每个 Agent 的 **Overview** 标签页里选默认模型和技能。不同 Agent 可以用不同模型——轻量任务用 Haiku，复杂任务用 Opus。

对话时用输入框里的 Agent 选择器切换。
