## What is MCP?

MCP (Model Context Protocol) is an open protocol standard that lets AI assistants securely connect to external tools, data sources, and services.

Think of MCP as a "plugin system" for AI. It lets AI:

  <Card title="Team Chat" icon="comments">

  <Card title="Note Management" icon="note-sticky">

  </Card>

  <Card title="Code Management" icon="github">
    Manage GitHub repositories and Issues
  </Card>

    Connect to 1000+ other apps and services
  </Card>

## Why Use MCP?

After installing MCP in Happycapy, you can:

### Automate Routine Tasks
Let AI handle emails, schedules, and document organization

### Cross-Platform Collaboration
Do multiple application operations in one conversation (like reading attachments from Gmail, processing them, and uploading to Notion)

### Simplify Development
Manage GitHub repositories, create Issues, and view code directly in conversations

## Configure MCP in Happycapy

Happycapy supports custom MCP server configuration. You can add any MCP-compatible service — GitHub, Notion, Slack, Gmail, and more — directly within your Happycapy session.
