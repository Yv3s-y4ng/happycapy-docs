# 图片和视频教程

## 生成图片

直接描述你想要的图片，AI 会帮你选择最合适的模型生成。

### 基础用法

```
帮我生成一张图片：一只可爱的水豚在温泉里泡澡
```

### 指定风格

```
用油画风格画一幅日落下的山脉，色调温暖
```

### 指定模型

支持 50+ 种模型，常用的有：

| 模型 | 特点 |
|------|------|
| FLUX | 通用高质量，适合大多数场景 |
| Gemini | Google 出品，擅长写实风格 |
| Grok | xAI 出品，创意性强 |

```
用 FLUX 模型生成：一个未来城市的鸟瞰图，赛博朋克风格
```

### 编辑图片

上传一张图片后，你可以让 AI 修改：

```
把这张图片的背景换成白色
把图片分辨率提高到 4K
去掉图片右下角的水印
```

## 生成视频

### 文字生成视频

```
根据以下描述生成一个 15 秒视频：
一只水豚慢慢走进温泉，周围是雪景，蒸汽升起
```

### 支持的视频模型

| 模型 | 特点 |
|------|------|
| Veo 3 | Google 出品，画质最高 |
| Seedance | 动态效果好 |
| Wan | 中文理解力强 |

### 视频编辑

上传视频后可以做：

```
把这个视频的前 10 秒剪出来
给视频加上中文字幕
把两个视频拼接在一起
把视频转成 GIF
```

<Tip>
视频生成需要一点时间，通常 30 秒到 2 分钟。AI 会告诉你进度。
</Tip>
