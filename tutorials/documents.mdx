# 文档和数据教程

## 写文档

### 写报告

```
帮我写一份 Q1 季度工作总结，包含以下要点：
- 完成了用户系统重构
- 上线了支付功能
- 用户增长 30%
```

### 写文章

```
写一篇 2000 字的博客文章，主题是"2026 年 AI 发展趋势"，
语气专业但不枯燥，面向技术从业者
```

### 生成 PDF

```
把上面的文章导出成 PDF，用专业的排版
```

## 处理表格

### 创建表格

```
帮我做一个项目进度跟踪表，包含：
任务名称、负责人、开始日期、截止日期、状态
```

### 分析 Excel

上传 Excel 或 CSV 文件后：

```
分析这个销售数据，告诉我：
1. 每月销售趋势
2. 最畅销的产品 Top 5
3. 做成柱状图和折线图
```

## 数据可视化

```
用这组数据做一个好看的图表，要适合放在 PPT 里
```

AI 支持生成：
- 柱状图、折线图、饼图
- 热力图、散点图
- 交互式图表（HTML）
- 导出为 PNG/PDF

## 做 PPT

```
把这篇文章做成 15 页的演示文稿：
- 第 1 页：标题页
- 第 2-3 页：背景介绍
- 第 4-10 页：核心内容（每页一个要点）
- 第 11-14 页：数据和图表
- 第 15 页：总结
```

<Tip>
你可以上传已有的 PPT 让 AI 修改，也可以从零开始生成。
</Tip>
