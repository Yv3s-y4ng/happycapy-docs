# 网页和设计教程

## 做落地页

```
帮我做一个 SaaS 产品的落地页：
- 产品名：TaskFlow
- 定位：团队任务管理工具
- 主题色：紫色
- 包含：Hero 区域、功能介绍、价格方案、FAQ
```

AI 会生成一个完整的 HTML 页面，你可以直接预览和下载。

## 做设计稿

### 海报

```
做一张活动海报：
- 主题：AI 技术大会 2026
- 时间：3 月 15 日
- 地点：上海世博中心
- 风格：科技感、深色背景
```

### 社交媒体图片

```
做一张适合小红书的封面图，主题是"5 个提高效率的 AI 工具"，
风格简洁，有吸引力
```

### 名片和 Logo

```
帮我设计一个简约风格的 Logo，品牌名是 HappyCapy，
元素可以用水豚和科技感的线条
```

## 交互式设计

AI 可以生成带交互效果的网页：

```
做一个产品展示页，包含：
- 顶部导航栏
- 滚动动画效果
- 点击展开的 FAQ 手风琴
- 底部联系表单
```

## 3D 效果

```
做一个 3D 旋转的产品展示，用 Three.js，
产品是一个手机，可以鼠标拖动旋转查看
```

<Tip>
生成的网页可以直接部署使用，也可以下载源代码二次开发。
</Tip>
