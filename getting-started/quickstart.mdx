## Your first conversation

Don't be nervous — just talk to happycapy like you would a friend.

In the input box at the bottom, try this:

```
Draw a cute capybara in cartoon style
```

Press Enter and watch happycapy create your image.

Or try:

```
Generate cute capybaras in different art styles
```

happycapy will adjust to your requirements. You'll see the images in the `outputs/` folder and can download them. The copyright is entirely yours.

## Working with files

### The uploads folder

Drag files directly into the conversation window and happycapy will process them.

<Note>
  **Supported formats**: Images (JPG/PNG), Documents (PDF/Word/Excel), Videos (MP4), Data (CSV/JSON), Code (all languages)

  **Maximum file size**: 100MB per file

  **Batch uploads**: Multiple files at once
</Note>

## Quick examples to get started

<AccordionGroup>
  <Accordion icon="image" title="Case 1: Generate and download images">
    The simplest example — get started in 10 seconds.

    **Step 1: State your needs**

    Type in the input box:
    ```
    Generate an image: a cute capybara sunbathing on grass
    ```

    **Step 2: Wait for generation**

    The image will appear in the conversation area in about 10-15 seconds and automatically save to the `outputs/` folder.

    **Step 3: Download locally**

    - **Method 1**: Find the image in the right file list, right-click to download
    - **Method 2**: Simply say "package this image for me to download"

    <Tip>
      Not satisfied? Just say "Generate another one with a more cartoon style" and happycapy will adjust accordingly.
    </Tip>
  </Accordion>

  <Accordion icon="file-arrow-up" title="Case 2: Upload and process files">
    Learn to upload files so happycapy can help you process various documents.

    **Step 1: Upload files**

    - **Method 1**: Drag and drop files directly into the conversation window
    - **Method 2**: Click the + button next to the input box to select files

    **Step 2: Tell happycapy what to do**

    ```
    Help me organize this Excel file:
    1. Remove duplicate data
    2. Sort by date
    3. Generate a clean new spreadsheet
    ```

    **Step 3: Wait for processing**

    happycapy will read the file, process the data, generate new files, and explain what changes were made. Processed files are saved to the `outputs/` folder.
  </Accordion>
</AccordionGroup>
