## The interface

Visit [happycapy.ai](https://happycapy.ai), register, and log in.

<Frame>
  <img src="/images/features/main-interface.png" alt="Happycapy main interface" />
</Frame>

- **Sidebar** — Create Desktops, browse the Skill Store, manage Files, set up Automations.
- **Conversation area** — Talk to Happycapy, see results in real time.
- **Input box** — Type your request, attach files with **+**, pick a Skill, or choose a model.

## Start a conversation

Type what you need in the input box. Happycapy handles the rest.

```
Build me a personal portfolio website with a dark theme
```

```
Analyze this CSV and create a chart showing monthly trends
```

```
Generate a logo for a coffee shop called "Morning Brew"
```

Outputs are saved to your sandbox. Find them in **Files** in the sidebar.

## Upload files

Drag files into the conversation window, or click **+** in the input box.

Happycapy can read and process them immediately — spreadsheets, PDFs, images, code, video.

<Note>
  Supported formats: Images (JPG/PNG), Documents (PDF/Word/Excel), Videos (MP4), Data (CSV/JSON), Code (all languages). Maximum 100MB per file.
</Note>

## Install a skill

Skills extend what Happycapy can do. Open **Skill Store** in the sidebar, find a skill, and click install.

Once installed, Happycapy uses it automatically when relevant — or you can call it by name.

```
/frontend-design Create a landing page for a SaaS product
```
