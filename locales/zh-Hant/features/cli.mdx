## 什麼是 CLI？

CLI（命令列介面）是一種透過文字指令操作軟體的工具。與網頁或 App 不同，CLI 在終端機裡執行，一行指令就能完成操作。

Happycapy 基於雲端沙箱執行，具備**完整的 Linux 終端機能力**。安裝好的 CLI 在整個環境中全域可用，無需擔心本地依賴衝突。你用自然語言說需求，Happycapy 自動調度對應 CLI 完成操作——對話與命令列無縫融為一體。

## 如何開始

找到你想用的 CLI，把以下格式的 prompt 發給 Happycapy：

```
安裝這個：[倉庫地址]，全域安裝，並告訴我有哪些功能，然後引導我完成設定
```

Happycapy 會自動完成安裝，介紹功能，然後引導你互動式完成設定。
