## 概覽

Agent 是自訂的 AI 角色，擁有獨立的身分、記憶和技能組合。預設 Agent 是 **Happycapy**。你可以為不同工作流建立不同的 Agent——研究助手、程式碼審查員、寫作夥伴——各有各的性格和上下文。

## Agent 檔案

</Frame>

每個 Agent 透過一組 markdown 檔案來定義自己：

- **SOUL.md** — 核心價值觀和原則
- **USER.md** — 關於你（被服務者）的資訊
- **IDENTITY.md** — Agent 的角色和性格
- **MEMORY.md** — 跨會話的持久記憶
- **AGENTS.md** — 主指令檔案，把所有東西串起來

每次開始對話時，Agent 按順序讀取這些檔案——先知道自己是誰，再開始工作。

## 配置 Agent

不用手寫這些檔案。在側邊欄的 **Agents** 裡建立新 Agent，然後開一個對話，選定這個 Agent，直接說：

```
幫我配置這個 Agent。
```

Happycapy 會問你需要什麼，然後自動產生所有配置檔案。描述角色、你的偏好、你希望它記住的東西——Happycapy 搞定剩下的。

## 模型和技能

在每個 Agent 的 **Overview** 標籤頁裡選預設模型和技能。不同 Agent 可以用不同模型——輕量任務用 Haiku，複雜任務用 Opus。

對話時用輸入框裡的 Agent 選擇器切換。
