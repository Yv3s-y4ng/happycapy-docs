## 介面

前往 [happycapy.ai](https://happycapy.ai)，註冊並登入。

<Frame>
  <img src="/images/features/main-interface.png" alt="Happycapy 主介面" />
</Frame>

- **側邊欄** — 建立 Desktop、瀏覽 Skill Store、管理檔案、設定自動化。
- **對話區** — 和 Happycapy 對話，即時查看結果。
- **輸入框** — 輸入需求，用 **+** 上傳檔案，選擇 Skill 或切換模型。

## 開始對話

在輸入框裡寫下你的需求，Happycapy 處理剩下的。

```
幫我建一個深色主題的個人作品集網站
```

```
分析這份 CSV，生成按月趨勢的圖表
```

```
為一家叫「Morning Brew」的咖啡店設計 logo
```

輸出檔案保存在沙箱裡，點側邊欄的 **Files** 查看。

## 上傳檔案

把檔案拖進對話視窗，或點輸入框的 **+** 選擇檔案。

Happycapy 可以直接讀取和處理——表格、PDF、圖片、程式碼、影片都行。

<Note>
  支援格式：圖片（JPG/PNG）、文件（PDF/Word/Excel）、影片（MP4）、資料（CSV/JSON）、程式碼（所有語言）。單檔最大 100MB。
</Note>

## 安裝技能

技能擴展 Happycapy 的能力。打開側邊欄的 **Skill Store**，找到技能，點擊安裝。

安裝後 Happycapy 會在需要時自動調用，你也可以直接用名字調用：

```
/frontend-design 做一個 SaaS 產品的落地頁
```
