## 界面

访问 [happycapy.ai](https://happycapy.ai)，注册并登录。

<div style={{ background: '#F8AA62', borderRadius: '12px', padding: '24px' }}>
  <img src="/images/features/main-interface.png" alt="Happycapy 主界面" style={{ borderRadius: '8px', maxWidth: '100%', display: 'block', margin: '0 auto' }} />
</div>

- **侧边栏** — 创建 Desktop、浏览 Skill Store、管理文件、设置自动化。
- **对话区** — 和 Happycapy 对话，实时查看结果。
- **输入框** — 输入需求，用 **+** 上传文件，选择 Skill 或切换模型。

## 开始对话

在输入框里写下你的需求，Happycapy 处理剩下的。

```
帮我建一个深色主题的个人作品集网站
```

```
分析这份 CSV，生成按月趋势的图表
```

```
为一家叫"Morning Brew"的咖啡店设计 logo
```

输出文件保存在沙盒里，点侧边栏的 **Files** 查看。

## 上传文件

把文件拖进对话窗口，或点输入框的 **+** 选择文件。

Happycapy 可以直接读取和处理——表格、PDF、图片、代码、视频都行。

<Note>
  支持格式：图片（JPG/PNG）、文档（PDF/Word/Excel）、视频（MP4）、数据（CSV/JSON）、代码（所有语言）。单文件最大 100MB。
</Note>

## 安装技能

技能扩展 Happycapy 的能力。打开侧边栏的 **Skill Store**，找到技能，点击安装。

安装后 Happycapy 会在需要时自动调用，你也可以直接用名字调用：

```
/frontend-design 做一个 SaaS 产品的落地页
```
