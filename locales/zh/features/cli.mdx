## 什么是 CLI？

CLI（命令行界面）是一种通过文字命令操作软件的工具。与网页或 App 不同，CLI 在终端里运行，一行命令就能完成操作。

Happycapy 基于云端沙箱运行，具备**完整的 Linux 终端能力**。安装好的 CLI 在整个环境中全局可用，无需担心本地依赖冲突。你用自然语言说需求，Happycapy 自动调度对应 CLI 完成操作——对话与命令行无缝融为一体。

## 如何开始

找到你想用的 CLI，把以下格式的 prompt 发给 Happycapy：

```
安装这个：[仓库地址]，全局安装，并给我讲讲有哪些功能，然后引导我完成配置
```

Happycapy 会自动完成安装，介绍功能，然后引导你交互式完成配置。
