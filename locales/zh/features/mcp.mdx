## 什么是 MCP?

MCP(Model Context Protocol,模型上下文协议)是一个开放的协议标准,让 AI 助手能够安全地连接到各种外部工具、数据源和服务。

简单来说,MCP 就像是给 AI 装上了"插件系统",让它能够:

  <Card title="邮件集成" icon="envelope">
    读写你的 Gmail 邮件
  </Card>

  <Card title="团队聊天" icon="comments">
    在 Slack 上发送消息
  </Card>

  <Card title="笔记管理" icon="note-sticky">

  </Card>

  <Card title="代码管理" icon="github">

  </Card>

  <Card title="广泛集成" icon="plug">
    连接 1000+ 其他应用和服务
  </Card>
</CardGroup>

## 为什么需要 MCP?

在 Happycapy 中安装 MCP 后,你可以:

### 自动化日常工作
让 AI 帮你处理邮件、安排日程、整理文档

### 跨平台协作
在一个对话中完成多个应用的操作(比如从 Gmail 读取附件,处理后上传到 Notion)

### 简化开发流程
直接在对话中管理 GitHub 仓库、创建 Issue、查看代码

## 在 Happycapy 中配置 MCP

Happycapy 支持自定义 MCP 服务器配置。你可以直接在会话中添加任何兼容 MCP 的服务——GitHub、Notion、Slack、Gmail 等。
