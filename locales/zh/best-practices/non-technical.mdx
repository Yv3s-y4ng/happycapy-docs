### 场景 2: 快速生成营销素材

**真实案例:** 增长营销团队用 Happycapy 在 15 分钟内生成数百个广告文案变体。

#### 在 Happycapy 中的做法

提供详细要求进行批量创建:

```
帮我创建 20 个 Google Ads 广告变体:
- 产品: AI 协作工具 Happycapy
- 目标受众: 中小企业主
- 卖点: 零技术门槛、提升工作效率、云端运行
- 标题限制: 30 字符以内
- 描述限制: 90 字符以内
```

#### Happycapy 会做什么

1. 生成多个创意方向
2. 自动检查字符限制
3. 输出为可以直接导入广告平台的 CSV

**节省时间: 从 2 小时 → 15 分钟**

## 非技术人员的建议

### 1. 从简单任务开始建立信心

<AccordionGroup>
  <Accordion icon="image" title="首先尝试: 基础创作">
    尝试简单任务: "帮我生成一张海报"
  </Accordion>

  <Accordion icon="chart-line" title="然后尝试: 数据分析">
    转向数据任务: "分析这些数据并制作图表"
  </Accordion>

  <Accordion icon="diagram-project" title="最后尝试: 复杂工作流">
    处理更复杂的多步骤工作流
  </Accordion>
</AccordionGroup>

### 2. 像和助理聊天一样描述需求

<Columns cols={2}>
  <div>
    **不要想:**

    "我应该用什么工具?"
    "我该如何编程实现?"
  </div>
  <div>
    **只需说:**

    "我想要什么结果?"
    "我的最终目标是什么?"
  </div>
</Columns>

<Tip>
  Happycapy 会自动为你选择合适的工具和模型。
</Tip>

### 3. 保存成功的工作流

当你发现一个效果好的提示时:

1. **记录有效的提示** 到你的笔记中
2. **对于重复任务**,直接复制上次的描述
3. **Pro/Max 用户** 可以设置自动化按计划运行任务

### 4. 不要害怕问"傻"问题

<Note>
  - Happycapy 不会评判你
  - 试错是学习的一部分
  - 每次对话都是独立的,随时可以重新开始
</Note>

## 真实案例

### 财务: 月度预算报告

```
帮我创建月度预算 vs 实际支出报告:

1. 读取 budget.xlsx 和 spending_january.xlsx
2. 计算每个类别的差异
3. 将超出预算的类别标记为红色
4. 创建显示以下内容的可视化图表:
   - 预算 vs 实际对比
   - 预算使用百分比
5. 生成一页的执行摘要
```

### 法务: 合同分析

```
分析这 5 份供应商合同并创建对比表:

[上传合同文件]

请提取并对比:
- 合同期限
- 付款条款
- 责任条款
- 终止条件
- 关键义务

创建一个汇总表,帮助我做出决策。
```

### 运营: 任务管理

```
处理我的每周团队会议纪要:

[粘贴会议记录]

请:
1. 识别所有行动事项
2. 根据上下文提到的人员分配
3. 提取截止日期(或建议合理的日期)
4. 创建包含以下列的表格: 任务、负责人、截止日期、状态
5. 导出为 CSV 以便导入我们的项目管理工具
```

### HR: 候选人筛选

```
帮我筛选这 20 份简历:

[上传简历文件]

职位要求:
- 3 年以上项目管理经验
- PMP 认证优先
- 有敏捷方法论经验
- 强沟通技巧

请:
1. 按资质匹配度对候选人排名
2. 突出每个人的关键优势和差距
3. 创建前 5 名候选人的候选名单
4. 根据他们的背景生成面试问题
```

## 成功技巧

### 从小做起并逐步扩展

不要试图一次自动化所有事情。相反:

1. **选择一个重复性任务** 每周占用 30 分钟以上
2. **用 Happycapy 自动化该任务**
3. **通过 2-3 次迭代优化工作流**
4. **一旦第一个任务运行顺畅,转向下一个任务**

### 建立你的提示库

<Tip>
  创建一个文档,按任务类型组织你表现最好的提示。这成为你的个人自动化工具包。
</Tip>

### 利用模板

一旦你创建了成功的工作流:

- 保存提示结构
- 每次替换具体细节(日期、文件名等)
- 与你的团队共享成功的模板

### 请求解释

如果你想了解 Happycapy 在做什么:

```
请解释你处理这些数据的每个步骤
以及为什么选择这种方法。
```

这有助于你学习并改进未来的提示。
